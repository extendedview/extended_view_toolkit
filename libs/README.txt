libs
====

This folder contains external libraries needed for some features of the toolkit:


GIL
---
Geometry Interaction Library.
GIL is a set of Pure Data abstractions, aiming at the interaction of geometric objects or mouse cursor inside GEM.
Copyright (C) 2011  Marian Weger, GPL v3
https://github.com/m---w/gil


Kollabs
-------
Collected abstractions for absolute data collapse.
KOLLABS is an abstraction library for Pure Data, that covers data management, OSC communication and state saving.
Copyright (C) 2012  Marian Weger, GPL v3
https://github.com/m---w/kollabs
